在线买煤功能使用说明 

1、打开拉煤宝软件，点击在线买煤。


2、挑选符合需求的煤炭。在煤炭列表中根据热量和价格选择合适的煤，点击此条煤炭信息可查看该煤炭的详细指标，若指标不全或没有需要了解的指标信息，可点击联系人右边的电话按钮，通过电话联系卖家获取详细指标信息。


3、部分煤价信息属于精品信息，精品信息需付费1~2元，您可选择代金券或微信支付来查看精品信息。（您支付的费用是向发布该精品信息的信息部支付的，并不是向平台支付）


4、当您觉得指标符合价格合适，有意向购买该煤炭时，点击煤炭详情里的“我要买煤”，输入您的预订量，选择是否派车提煤（是则买家自己派车，否则委托信息部找车），信息填写完毕后点击右上角的提交，卖方就能受理您的买煤订单。（订单发出后将由信息部自己确认是否接单）


5、.若信息部拒绝受理则交易不成立，若信息部受理订单，则双方磋商后确认订单成立，订单成立后，若买家派车自提，则进入派车页面
