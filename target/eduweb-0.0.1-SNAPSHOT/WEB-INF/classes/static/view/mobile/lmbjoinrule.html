拉煤宝平台入驻规则

申请标准

      矿口企业及煤炭供应商（以下简称矿口企业）均可申请加入拉煤宝平台，但应符合以下入驻标准。

企业供应商申请标准：

      持有完成最近年度年检的《营业执照》，营业范围在平台服务领域，且拟在公司平台开展的经营活动不超过其《营业执照》核准的经营范围；

      持有煤炭类生产或经营许可证等材料；

      无不良行为记录（包括但不限于买卖投诉、纠纷等）；

入驻基本流程

      在拉煤宝官方网站下载。企业加入审批表.docx

      填写申请表。

      将填写好的申请表发送至Johnpang@sxhalo.com 。

审  核

      1.拉煤宝平台对矿口企业线上提交的申请表进行审核，一般3-5个工作日通知审核结果。

      2.拉煤宝平台可根据实际情况要求矿口企业提供进一步证明材料，矿口企业应配合提供。

      3.拉煤宝平台有权视情况对矿口企业进行实地考察认证，实地认证一般7-15个工作日通知审核结果。

入驻成功

      审核通过的矿口企业，则分配企业用户账号并安装、培训，矿口企业上传相关资料后正式交付使用，入驻成功。

入驻企业权利和义务：

      (1)入驻矿口企业应当具有相关行政部门颁发的《营业执照》、《税务登记证》、《组织机构代码证》及各类商品经营许可证明，同时矿口企业保证提交给拉煤宝平台的上述相关证明材料真实、合法、可靠，并对其真实性负责。

      (2) 入驻矿口企业应承诺在利用拉煤宝平台开展煤炭销售等商务活动时严格遵守《中华人民共和国消费者权益保护法》等相关法律法规，尊重消费者的隐私权和其他法定权利。同时严格遵守拉煤宝平台与其他关联方所签协议书中拉煤宝平台需要承担的与消费者（客户）权益相关的义务和责任，但是，拉煤宝平台应该事先将相关文本的复印件交给入驻矿口企业。

      (3)入驻矿口企业通过拉煤宝平台发布的所有产品品质必须符合国家质量技术标准或行业标准，同时取得相关行政部门出具的产品合格证，若产品出现质量问题，矿口企业须承担因此而造成的一切损失(包括但不限于:向消费者赔礼道歉、退货、赔偿损失和赔偿拉煤宝平台的损失)。

      (4) 矿口企业负责平台用户网上预约之后的所有后续服务（包括但不限于发货、收款、退货、退款）。矿口企业在接到拉煤宝平台的预约订单后，应及时确认订单的有效性，尽快发货并确保平台用户在最短时间内能收到货品。

      (5) 矿口企业保证其发布在拉煤宝平台上的信息合法、真实和有效，并且保证不存在任何虚构、错误及误导的信息，否则矿口企业将承担全部的经济损失及法律责任。

      (6)在相关业务的宣传工作中，矿口企业不得损害拉煤宝平台的名称权、名誉权等非财产权利。拉煤宝平台有权对矿口企业的宣传内容进行事先审核。对于拉煤宝平台的任何修改要求和建议，矿口企业应该立即接受并改正。

      (7) 矿口企业应该按照拉煤宝平台的要求为履行本协议而使用计算机，不得以任何形式对拉煤宝平台的计算机环境进行破坏，由于矿口企业的原因造成拉煤宝平台的计算机环境出现故障，矿口企业有义务配合拉煤宝平台进行系统的恢复工作并承担所需费用和损失赔偿责任。

      (8) 矿口企业应妥善解决与消费者之间的纠纷，并承担因此而产生的责任，拉煤宝平台将对此给予必要的协助。
