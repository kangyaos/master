保证金说明 

保证金是买煤求购中，保证买家或信息部用户提供的消息的真实性和有效性的担保，买家在发布买煤求购信息的时候可以自由选择是否发布带保证金的买煤求购信息，若买家发布不带保证金的买煤求购信息，则信息部提交意向时不用缴纳保证金。若买家发布了带保证金的买煤求购信息，则信息部在提交意向的时候需缴纳同等金额的保证金。

1.若买家发布的买煤求购在没有信息部提交意向时取消发布，则扣除2%的管理费后退款。

2.信息部若在买家未确认时取消投递意向，则扣除信息部2%的管理费后退款。

3.若买家取消发布有信息部提交意向的买煤求购信息，则扣除（2%+N*0.5%,N为提交意向的信息部数量）的管理费后退款，并对提交意向的信息部退款（免除管理费）。

4.若未达成意向，双方同意退款，则双方各扣除2%的管理费后退款，若一方申请退款后，另一方在24小时内未响应则扣除管理费后自动退款。

5.若未达成意向，一方申请退款后，另一方拒绝退款时，某一方申请平台介入后平台将在7*24小时内介入并处理。平台介入后双方退还的保证金和扣除的管理费以及平台介入的手续费，根据实际情况决定。

6.若双方达成意向，则在双方确认意向后，系统将在24小时内扣除买家（2%+N*0.5%,N为提交意向的信息部数量）的管理费后退款。
