代金券的类型和规则 

        当前在拉煤宝平台上可使用的代金券类型有：全平台券；专有券；满减代金券；无门槛代金券。 


在支付资讯费时：

1、全平台券能够在全平台使用；

2、专有券针对特殊煤炭品类或货运路线的代金券、针对特殊煤炭商品或货运信息的代金券、针对特殊活动的代金券、针对特殊用户的代金券；

3、满减代金券在当单笔金额满足减免条件后可以使用；

4、无门槛代金券是不限支付金额就可以使用的代金券；

5、每张代金券的适用范围会在该张代金券上标明。


其它使用细则

1、代金券适用于拉煤宝平台用户在拉煤宝APP上支付资讯费时按代金券规则使用，不能进行兑现或其他用途。

2、不得以不正当手段（作弊、恶意套现、虚假交易、扰乱系统、实施网络攻击等）获取或使用代金券。

3、当使用了代金券的交易发生争议后，拉煤宝平台将酌情对已使用的代金券进行处理。拉煤宝平台有权视情况选择冻结或作废相关用户持有的代金券和取消相关用户的代金券使用资格。

4、用户同意按照代金券的规则使用各类代金券，代金券中未说明的规则，如本使用须知有规定的，对代金券的使用亦产生效力；代金券中未说明的规则或与本使用须知不一致时，以本使用须知为准。

5.已支付抵用的代金券金额超出部分不予退还（例：精品信息费用1.5元，使用两张1元代金券，超出0.5元部分不予退还）
