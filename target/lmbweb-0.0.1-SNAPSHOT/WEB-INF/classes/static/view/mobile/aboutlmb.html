“三大服务、四大优势”
铸就全新“互联网+煤炭”平台



拉煤宝三大服务功能

        查货源：货源质量优、更新快。物流价格准确、透明。货运机会多、利润大。煤矿导航，全面覆盖。避免空车上路，降低成本。服务好，不收费。

        知煤讯：把控行业走势。信息实时更新。价格准确透明。

        找司机：货流追踪；预知市场走向；优化产业结构；提高精细化销售；定位销售漏洞，解决库存压力。


拉煤宝具备了四大优势

        政策优势：制定“互联网+”行动计划是国家“稳增长、调结构”行动中的一环，它的出现，将打破煤炭行业的垄断，为煤炭行业带来新的机遇。不仅增强了煤企对市场的判断力，而且在煤炭销售商会出现大的变革，可以根据用户需要生产多样化的产品，并能省去许多中间环节，快捷方便，省时省力。

        行业优势：拉煤宝技术运营团队深耕煤炭行业多年，以行业转型为契机，与产业链上下游的优势企业进行融合，通过不断的科技创新，延伸和完善原有的产业结构，开发出更加符合市场需求的产品。

        团队优势：陕西光环网络科技有限公司是一家从事软件开发和网络通讯工程的高新技术企业，陕西首家交易平台系统开发商，光环网络顺应行业发展趋势，借助国家“互联网+”发展战略，充分发挥“互联网+”、大数据、金融三大优势，致力于将创痛煤炭供应产业链完全激活，创造一个崭新的数万亿市场。

        资本优势：光环科技以其强大的资金储备与顶尖的技术为“拉煤宝”各时期的建设提供坚实基础。

        在整个行业生存环境发生巨大变化的时候，陕西光环网络科技有限公司推出“拉煤宝”APP，布局互联网平台，利用移动互联网为传统煤炭市场转型做了一次大胆、富有远见的尝试，是对煤炭市场未来发展方向的一次大胆探索，也为整个行业的发展指出了方向。
