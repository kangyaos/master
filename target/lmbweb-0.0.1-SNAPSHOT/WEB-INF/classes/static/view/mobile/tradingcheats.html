交易秘籍 

一、预约煤炭

        可以选择城市地区浏览煤炭产量、价格、发热量等最新资讯信息，也可以浏览优惠情况和排名情况，同时可以直接下单预约买煤。预约的方式有两种：

（1）电话预约，直接拨打煤炭企业联系电话进行预约；

（2）App预约，先注册登录，登录后直接填写预约信息，提交预约单。等待煤炭企业进行受理，受理成功后会有系统消息提示，也可以在 “我的预约单”中查可产看所提交的预约信息。

注：填写预约信息时，交易量和预约时间为必填项，车牌号为选填项。


二、货运抢单

        最新货运信息可在货运列表中直接浏览，也可通过搜索地点来查找，同时可以直接进行抢单拉货。抢单流程：

        一是必须先注册登录，并进行车辆认证，认证后进行抢单。

        二是等待煤炭企业审核，审核成功会有系统消息提示，并可在“我的货运单”中随时查看货运单信息；审核未通过，系统消息会提示未通过原因，可进行再次抢单；审核过程中可以取消抢单，确认取消将不可再抢此单，并且24小时不能再抢任何货运单。

        三是抢单成功后（因为自身原因未能在出发前到），煤炭企业可取消此单，可在“我的货运单”中查看此单信息。

注：车辆未成功认证，或是已经提交抢单申请等待审核，此时不能抢单，否则会提示抢单失败。
